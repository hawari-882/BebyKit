Subtitle Headings

Smaller headings (subtitle) for heading. Add subtitle inside the class attribute. Take a note that this subtitle work only if the previous heading has title class!
To implement it, use the same heading or lower (by one level).
See examples below: 

<h1 class="title">
    Hello There Title
</h1>

<h1 class="subtitle">
    Hello There Subtitle
</h1>

Hello There Title

Hello There Subtitle

Or...

<h1 class="title">
    Hello There Title
</h1>

<h2 class="subtitle">
    Hello There Subtitle
</h2>

Hello There Title

Code Writing

The font of the code that is used in this framework, is monospace font.
Apply code writing by using semantic HTML tags.
See visual & code examples below:
code

console.log("Hello CSS"); console.log("Hello JS"); console.log("Hello BebyKit"); 
<code>console.log("Hello CSS");
console.log("Hello JS");
console.log("Hello BebyKit");</code>

pre

console.log("Hello CSS"); console.log("Hello JS"); console.log("Hello BebyKit"); 
<pre>console.log("Hello CSS");
console.log("Hello JS");
console.log("Hello BebyKit");</pre>

The differences between code and pre are on their default background color. You can change this by using Background Color Utility.
