Get Started

BebyKit is a powerful tool for front-end development, responsive and soft. Using un-omitted classes name and easy-to-remember classes name to make ease of use. Only 1 CSS file to use BebyKit!

Only 1 CSS file to use BebyKit!

CDN

Copy and paste this code below inside your project to get started. 

<button class="button button-outline button-passive">mini</button>

Breakpoints

BebyKit's built-in media breakpoints which you can use freely, responsive to its viewport window sizes.

Class Device(s) Size
mini Mobile ~ 576px
small Mobile ~ Tablet 577px ~ 768px
medium Tablet ~ Laptop 769px ~ 992px
large Laptop ~ Desktop 993px ~ 1200px
maxi Desktop ~ ( larger dimension ) 1201px ~
