Display

Manipulate HTML element display to suits your styles.

Class CSS 
display-block
 
.display-block {
    display: block;
}
 
display-inline
 
.display-inline {
    display: inline;
}
 
display-inline-block
 
.display-inline-block {
    display: inline-block;
}
