Icons

For now, BebyKit uses icons that licensed by others, and now BebyKit is looking for a new icon and own-licensed icon for BebyKit. You can contribute to BebyKit open-source project by designing and create a new icons for BebyKit. It would be helpful for BebyKit's development.

If you have any ideas, please discuss in GitHub or other ways in online social community like Gitter.
